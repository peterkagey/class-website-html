Vieta's formulas

If we have a monic polynomial $$x^n + a_{n-1}x^{n-1} + \dots + a_{2}x^2 + a_1x + a_0$$ with complex roots $r_1, r_2, \dots, r_n$, then $$\begin{align}a_0 &= r_1r_2 \cdots r_n\\ &\vdots \\ a_{n-2} &= r_1r_2 + r_1r_3 + \dots + r_1r_n + \dots + r_{n-1}r_n\\ a_{n-1}&= r_1 + r_2 + \dots + r_n.\end{align}$$
