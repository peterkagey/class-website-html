Polynomial interpolation

Faulhaber's formula

The function \(\displaystyle f(n) = \sum_{k=1}^n k^d\) is a degree-${(d+1)}$ polynomial.

Problem 1

Prove that if $p(k)$ is any degree-$d$ polynomial, then \(\displaystyle g(n) = \sum_{k=1}^n p(k)\) is a degree-${(d+1)}$ polynomial.

Lagrange interpolation

Given $(x_1, y_1), \dots, (x_n,y_n) \in \mathbb{R}^2

where $x_i \neq x_j$ for $i \neq j$, there exists a unique polynomial $p(x)$ with degree less than $n$ such that $p(x_i) = y_i$ for $1 \leq i \leq n$.

Problem 2

If $P(x)$ denotes a polynomial of degree $n$ such that \(\displaystyle P(k)=\frac{k}{k+1}\) for $k=0,1,2,\ldots,n$, determine $P(n+1)$.
