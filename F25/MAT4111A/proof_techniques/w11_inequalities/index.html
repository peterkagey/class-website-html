Inequalities

AM-GM inequality

The AM-GM (arithmetic mean/geometric mean) inequality states that if $x_1, x_2, \dots, x_n$ are nonnegative real numbers, then \[\frac{x_1 + x_2 + \dots + x_n}{n} \geq \sqrt[n]{x_1 \cdot x_2 \cdots x_n}.\]

Cauchy-Schwarz inequality

(See AoPS) For any real numbers $a_1, a_2, \dots, a_n$ and $b_1, b_2, \dots, b_n,$

\[\begin{align}&(a_1^2 + a_2^2 + \dots + a_n^2)(b_1^2 + b_2^2 + \dots + b_n^2) \\ &\qquad\geq (a_1b_1 + a_2b_2 + \dots + a_nb_n)^2.\end{align}\]

Problem 2

Let $a_1$, $a_2$, $\cdots$, $a_n$, $b_1$, $b_2$, $\cdots$, $b_n$ be positive real numbers such that $a_1 + a_2 + \cdots + a_n = b_1 + b_2 + \cdots + b_n$. Show that \[\frac {a_1^2}{a_1 + b_1} + \frac {a_2^2}{a_2 + b_2} + \cdots + \frac {a_n^2}{a_n + b_n} \geq \frac {a_1 + a_2 + \cdots + a_n}{2}.\]

1991 APMO, Problem #3

Problem 3
