Required Exercise 2 [2]

Suppose that $A_1, A_2, \dots A_{n+1}$ is a collection of sets and $f_1, f_2, \dots, f_n$ is a collection of functions such that $f_i \colon A_i \to A_{i+1}$ for all $i \in \{1, 2, \dots, n\}.$

Let $\circ$ denote the composition of functions so that $(g \circ h)(x) = g(h(x))$.

  1. Explain why $f_1 \circ f_2$ is not well-defined (i.e. doesn't make sense).
  2. Explain why $f_2 \circ f_1$ is well-defined, and state its domain and codomain.
  3. Prove that $f_n \circ f_{n-1} \circ \dots \circ f_2 \circ f_1$ is a well-defined function, and state its domain and codomain.

Required Exercise 3 [4]

Why do we care about equivalence relations? Because they partition our set. A partition of the set $X$ is a collection of (nonempty) subsets of $X$ such that every element of $X$ is in exactly one of the subsets. In particular, if we partition our set according to an equivalence relation $\sim$, then two elements $a, b \in X$ are in the same subset if and only if $a \sim b$. We call these "equivalent" subsets equivalence classes.

  1. If we look at the equivalence relation on the integers $\mathbb{Z}$, defined as $a \sim b$ if and only if $a \equiv b \pmod 3$, then we have three equivalence classes, which in abstract algebra we would describe with a slightly funny notation $$\begin{align}3\mathbb{Z} &= \{\dots, -6, -3, 0, 3, 6, \dots\} \\ 3\mathbb{Z} + 1 &= \{\dots, -5, -2, 1, 4, 7, \dots\} \\ 3\mathbb{Z} + 2 &= \{\dots\}.\end{align}$$ Here $3\mathbb{Z} + 1$ is "just notation": it's the name of a particular set. Briefly explain in words
    1. three examples of integers that are in $3\mathbb{Z} + 2$,
    2. how you determine which equivalence class an integer $n$ belongs to,
    3. how you know that every number is in exactly one of these equivalence classes.

In abstract algebra, we refer to the group of "integers modulo $3$" as $\mathbb{Z}/3\mathbb{Z}$, where the notation means that we think of everything in $3\mathbb{Z}$ as being the same. This process of "quotienting" a mathematical object is ubiquitous in abstract algebra, topology, linear algebra, and other upper-level mathematics, and it all depends on the notion of equivalence

  1. Now let $X = \{1, 2, 3, 4\}$ and let $$R= \{(1,1), (1,2), (1,4), (2,1), (2,2), (2,4), (3,3), (4,1), (4,2), (4,4)\},$$ which is to say that for $a, b \in X$, $a R b$ if and only if $(a, b) \in R$.
    1. Prove (or disprove) that $R$ is an equivalence relation.
    2. Describe the two equivalence classes of $X$.
  1. A relation which is not an equivalence relation does not partition a set. Suppose that $X$ is a set and $R_1, R_2,$ and $R_3$ are relations on $X$.
    1. Prove that if $R_1$ is not reflexive, then $R_1$ does not partition $X$ into equivalence classes.
    2. Prove that if $R_2$ is not symmetric, then $R_2$ does not partition $X$ into equivalence classes.
    3. Prove that if $R_3$ is not transitive, then $R_3$ does not partition $X$ into equivalence classes.
