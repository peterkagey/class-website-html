Choice Exercise 6 [4]

This is your programming problem for this week's homework.

The On-Line Encyclopedia of Integer Sequences (OEIS) is a database of mathematical sequences which currently contains 390048 sequences, with about 33 new sequences being add each day. One consequence of the structure of the database, is the even for "two-dimensional" sequences like Pascal's Triangle, we always have to input the data as a sequence. For Pascal's triangle, there's an obvious way to do it: read it by rows.

$$1 \\ 1 \quad 1 \\ 1 \quad 2 \quad 1 \\ 1 \quad 3 \quad 3 \quad 1 \\ 1 \quad 4 \quad 6 \quad 4 \quad 1$$ Thus OEIS sequence A007318 begins $$1, 1, 1, 1, 2, 1, 1, 3, 3, 1, 1, 4, 6, 4, 1, 1, 5, 10, 10, 5, 1, 1, 6, 15, 20, 15, 6, 1, \dots$$ However, for a sequence like A003987, we have to be a bit more thoughtful. In this sequence, we're trying to describe the "two-dimensional" table $T(n,k) = n \oplus k$ where $\oplus$ is the "bitwise XOR" operator, which is ubiquitous in low-level computer programming, and in Python is written n ^ k. $$\begin{align}&0 \quad 1 \quad 2 \quad 3 \quad 4 \quad 5\\ &1 \quad 0 \quad 3 \quad 2 \quad 5\\ &2 \quad 3 \quad 0 \quad 1\\ &3 \quad 2 \quad 1\\ &4 \quad 5\\ &5\end{align}$$ The way we do it, is that we read the table by antidiagonals, letting ${a(0) = T(0,0)}$, ${a(1) = T(0,1)}$, ${a(2) = T(1,0)}$, ${a(3) = T(0,2)}$, ${a(4) = T(1,1)}$, ${a(5) = T(2,0)}$, ${a(6) = T(0,3)}$, and so on.

  1. Write a program that prints out the first $50$ terms of the sequence, $a(0)$ through $a(49)$.
  2. Write a program that prints out the first $5151st$ term of the sequence, $a(5150)$.
  3. Write a program that prints out the $m$-th term of the sequence, but without computing any smaller term. In other words, given some integer $m$, figure out how to determine $n$ and $k$ such that $a(m) = T(n,k)$, and then use this to compute $a(m)$.

Choice Exercise 7 [4+4]

Do any subset of the following parts, but for each part you do, attempt all subparts.

  1. [4] In linear algebra, we write a existence and uniqueness proof whenever we prove that some set of vectors forms a basis for a vector space.
    1. Write the definition of a basis, spanning set, and linear independence.
    2. Write an example of a vector space $V$ together with a spanning set $S \subseteq V$ such that there is a vector $\vec v \in V$ with the property that $\vec v$ can be written as a linear combination of vectors in $S$ in multiple ways.
    3. Write an example of a vector space $V$ together with a subset $S \subseteq V$ such that there is a vector $\vec v \in V$ with the property that $\vec v$ cannot be written as a linear combination of vectors in $S$.
    4. Prove that the set $S = \{(1,2), (-1, 2)\}$ is a basis for $\mathbb{R}^2$ by proving that for every vector $\vec{v} \in \mathbb{R}^2$, there exists a unique way to write $\vec{v}$ as a linear combination of vectors in $S$.
    5. Prove that if $\mathcal{B}$ is a basis for a vector space $V$, and we either add or remove a vector from $\mathcal{B}$, then it is no longer a basis for $V$.
  2. [4] In abstract algebra, we can define the rational numbers in the following way.
    1. We define a relation on the set $X = \mathbb{Z} \times \left(\mathbb{Z} \setminus \{0\} \right)$ by saying that $(a, b) \sim (a', b')$ if and only if $ab' = a'b$.
      1. Prove that this relation is reflexive.
      2. Prove that this relation is symmetric.
      3. Prove that this relation is transitive.
    2. Now we introduce new notation: we write $\mathbb{Q} = X/\sim$, where $X/\sim$ means that we count elements that are related by $\sim$ as "the same". To explain why this makes sense, show that $\frac{a}{b} = \frac{a'}{b'}$ if and only if $(a, b) \sim (a', b')$.
    3. Show that defining $(a, b) + (a', b') = (ab' + a'b, bb')$ is analogous to how we define $\frac{a}{b} + \frac{a'}{b'}$.
    4. Show that defining $(a, b) \times (a', b') = (aa', bb')$ is analogous to how we define $\frac{a}{b} \times \frac{a'}{b'}$.

Choice Exercise 8 [4]

When I was a senior in college, I took the Putnam Exam for the first and only time. There was a question on the exam that—in a quite complicated and indirect way—led me to become a math professor.

  1. Set a timer, and spend $10$ minutes trying to make sense of what the problem is asking. In particular, try to explain why $f(3) = 8$ and $f(5) = 10$.
    Let $S$ be the set of all positive integers that are not perfect squares. For $n$ in $S$, consider choices of integers $a_1, a_2, \dots, a_r$ such that $n < a_1 < a_2 < \cdots < a_r$ and $n \cdot a_1 \cdot a_2 \cdots a_r$ is a perfect square, and let $f(n)$ be the minumum of $a_r$ over all such choices. For example, $2 \cdot 3 \cdot 6$ is a perfect square, while $2 \cdot 3$, $2 \cdot 4$, $2 \cdot 5$, ${2 \cdot 3 \cdot 4}$, ${2 \cdot 3 \cdot 5}$, ${2 \cdot 4 \cdot 5}$, and ${2 \cdot 3 \cdot 4 \cdot 5}$ are not, and so $f(2) = 6$. Show that the function $f$ from $S$ to the integers is one-to-one [an injection].
    2013 Putnam, Problem A2
  2. Prove that $f(5) = 10$, and prove that $f(8) \leq 15$ by giving a sequence of integers ${8 < a_1 < a_2 < 15}$ such that ${8 \cdot a_1 \cdot a_2 \cdot 15}$ is square.
  3. Set a timer, and spend $10$ minutes trying to prove that $f$ is an injection.
  4. After the timer goes off, find a solution writeup for the problem, read it until you understand it, then put the solution away, and write up a solution to the problem in your own words.

Choice Exercise 9 [3]

In this problem, you will try to spot a somewhat subtle error in a (false) "proof."

  1. Recall that given a set $A$, the power set $\mathcal{P}(A)$ is the set of all of its subsets. Write down the eight subsets of $\{1,2,3\}$ (and thus the eight elements of $\mathcal{P}(\{1,2,3\})$).
  2. Find the error in the following false proof.
    (False) proposition. There exists an injection $f \colon \mathcal{P}(\mathbb{N}) \to \mathbb{N}$.
    (False) proof. Let $p_i$ be the $i$-th prime, so $p_0 = 2$, $p_1 = 3$, $p_2 = 5$, $p_3 = 7$, $p_4 = 11$, and so on. Then define $f \colon \mathcal{P}(\mathbb{N}) \to \mathbb{N}$ as $$f(\{a_1, a_2, \dots, a_n\}) = p_{a_1}p_{a_2} \cdots p_{a_n},$$ where $f(\emptyset) = 1$. (For example, $f(\{0, 2, 4\}) = p_0p_2p_4 = 2 \cdot 5 \cdot 11 = 110$.)
    Because each distinct element $S \in \mathcal{P}(\mathbb{N})$ maps to a number with a distinct prime factorization, $f(S_1) = f(S_2)$ if and only if $S_1 = S_2$, therefore $f$ is injective (1-to-1).
  3. If there were such an injection, it would prove that the cardinality of $\mathcal{P}(\mathbb{N})$ is less than or equal to the cardinality of $\mathbb{N}$, and thus $\mathcal{P}(\mathbb{N})$ is countably infinite. However, it turns out that the cardinality of $\mathcal{P}(\mathbb{N})$ is the same as the cardinality of $\mathbb{R}$. Write a sentence or two about any thoughts, questions, or observations about this.
