[4 pts] Required Exercise 1.

In this exercise you're going to practice \(\LaTeX\). If you already have experience with \(\LaTeX\), you can instead choose items from Choice Exercise 4 below.

  1. Write something in both "inline math": 
    \(1 + 1 = 2\)

    and in "display style math":

    \[1 + 2 = 3\]
  2. Note that you can also type inline math and display style math respectively as 
    $1 + 1 = 2$

    and

    $$1 + 2 = 3$$

    Try it out, but keep in mind that it's considered better style to use the \(...\) and \[...\] commands.

  3. We use commands starting with \ to type most math symbols. Type \frac{3}{4} to get something that looks like \(\frac{3}{4}\). Does this look the same when you put it in inline math mode versus display style math?
  4. Often we want to write things with subscripts and superscripts (e.g. exponents). To do this in \(\LaTeX\), we go into math mode and type
    • a_1 to get \(a_1\) or
    • x^2 to get \(x^2\).
  5. Your online homework uses MathJax to display \(\LaTeX\), which is a JavaScript display engine. If you are on a computer and you right click on math that's been typeset with MathJax, you get a menu, and you can navigate to Show Math As > TeX Commands. Do this for some of the math on this page, copy it, and paste it into your \(\LaTeX\) document. This will be very helpful when you don't know how to typeset a symbol that appears on your homework.
  6. Use Detexify to draw a math symbol that you don't know the command for. Then use the code that it gives to type it in your document.
  7. There are lots of different "environments" that you can use. What happens if you put a sentence between \begin{proof} and \end{proof} commands like this: 
    \begin{proof}
  Proof goes here.
\end{proof}

    What happens if we use an itemize environment and type something like the following?

    A list of my favorite classes: \begin{itemize}
  \item MAT 3100W
  \item MAT 4170
  \item MAT 4180
\end{itemize}

    What happens if you use enumerate instead of itemize?

Prove the following:

  1. If the two numbers \(a_1\) and \(a_2\) are odd numbers, then their product \(a_1 \times a_2\) is an odd number.

  2. If the three numbers \(a_1\), \(a_2\), and \(a_3\) are odd numbers, then their product \(a_1 \times a_2 \times a_3\) is an odd number.

  3. If the four numbers \(a_1, a_2, \dots, a_4\) are odd numbers, then their product \(a_1 \times a_2 \times \dots \times a_4\) is an odd number.

  4. If the fifty numbers \(a_1, a_2, \dots, a_{50}\) are odd numbers, then their product \(a_1 \times a_2 \times \dots \times a_{50}\) is an odd number.

[5 pts] Choice Exercise 4.

You can do this in addition to or instead of Required Exercise 2. If you do it in place of that exercise, do 15 points of problems from the Choice Exercises section.

  1. Typing set symbols can be a little annoying sometimes, because you have to remember that { and } are special characters. To get around this, you need to escape them like S = \{1, 2, 3\}.
  2. In order to get the spacing right on set builder notation, it's helpful to use \mid instead of |. Try out both with \mid and with |, and describe the differences: \[\{n \in \mathbb{N} \mid \log(n) > 5 \}\]
  3. If you want to put parenthesis around some tall math, you'll want to use syntax like \left( and \right). Try this out by putting \int e^x dx in parenthesis with and without the left and right descriptors.
  4. Sometimes we want to put words in math mode. We can do this! Describe the similarities and differences between the following: 
    \[\{n \in \mathbb{N} \mid n is even \}\]
    \[\{n \in \mathbb{N} \mid n \text{is even} \}\]
    \[\{n \in \mathbb{N} \mid n \text{ is even} \}\]
  5. One \(\LaTeX\) command that I perhaps overuse is \underbrace{...}. Compare and contrast these three ways of expressing that we have a tuple containing \(k\) \(1\)s. Which do you like the best? 
    \[\big(\underbrace{1,1,\dots,1}_{k\text{ times}}\big)\]
    \[\left(\underbrace{1,1,\dots,1}_{k\text{ times}}\right)\]
    \[\underbrace{\left(1,1,\dots,1\right)}_{k\text{ times}}\]

[5 pts] Choice Exercise 5.

The factorial function \(n!\) is defined on the nonnegative integers: \[n! = n \cdot (n-1) \cdot (n-2) \cdots 3 \cdot 2 \cdot 1.\] For a few reasons, we also define \(0! = 1\).

There is a cool interpolating function called the Gamma function, which allows us to say in a meaningful way (!!) that "one half factorial" is half the square root of π!

\[\left(\frac{1}{2}\right)! = \frac{\sqrt{\pi}}{2}.\]

Here's how it works: we define

\[\Gamma(z) = \int_0^\infty t^{z-1} e^{-t} \mathrm{d} t\].

We want to prove that \(\Gamma(z) = (z-1)!\) for all integers \(z \geq 1\).

  1. Compute an integral to show that when \(z = 1\), \(\Gamma(1) = 0!\).
  2. Compute an integral to show that when \(z = 2\), \(\Gamma(2) = 1!\). (Hint: you might need to do integration by parts!)
  3. Compute an integral to show that when \(z = 2\), \(\Gamma(3) = 2!\).
  4. Prove that when \(z \geq 1\) is an integer, \(\Gamma(n) = (n-1)\Gamma(n-1)\).
    (Hint: you might need to do integration by parts!)
  5. Why can you conclude that \(\Gamma(z) = (z-1)!\) for all integers \(z \geq 1\)?

[≤8 pts] Choice Exercise 6.

Do any subset of the following parts.

In class, we started to discuss the problem of tiling a \(4 \times 5\) grid with each of the five free tetrominoes. (Recall that the five free tetrominoes are kind of like Tetris pieces: they are the different ways of putting four boxes together edge-to-edge, where two are considered the same if you can rotate or reflect one to the other.)

  1. [2 pts] How many ways can you tile the \(4 \times 5\) grid with each of the five free tetrominoes?
  2. [2 pts] Is it possible to tile the \(4 \times 10\) grid using each of the five free tetrominoes exactly twice?
  3. [4 pts] Draw the \(12\) pentominoes (those with five squares). Can these tile a \(12 \times 5\) grid OR a \(10 \times 6\) grid OR a \(4 \times 15\) grid? Take a picture or a screenshot and post it on the Discord showing why or why not.
