Here's another \(\LaTeX\) tip: occasionally it is useful to include images (such as screenshots, photos, or images you find on the internet) in your document.

  1. In order to include images, you'll want to add a package. Type \usepackage{graphicx} up near the other packages at the top of your file.
  2. Start by creating or finding the image you want to include, and make sure it's a .png or .jpg/.jpeg file type.
  3. Put the file in your "local directory." If you're using Overleaf, you'll use the same button that you used to upload your template in HW00:
  4. Make a note of the file's name and file extension, or rename it to something like test_image.png (making sure to keep the file extension the same as it was).
  5. Typically you want your image on a new line, so put a blank line in your .tex file, and below it type: 
    
  \noindent\includegraphics[width=\textwidth]{test_image.png}

[5 pts] Choice Exercise 8.

Only attempt this exercise if you're feeling quite confident about how proofs by induction work.

Here's a classic (incorrect) induction proof of the (obviously) false claim: "all horses are the same color." The proof has a subtle error in it. Can you spot it?

Claim: given any group of \(n\) horses, they all are the same color.

Proof: We proceed by induction, where the base case is clear: when \(n=1\), we have a single horse, and so all of the horses in that group are the same color.

Assume (as an induction hypothesis), that any group of \(n\) horses is the same color for all \(n = 1, 2, \dots, k\).

Suppose (as an induction hypothesis) that we have \(k+1\) horses, and we label them \(h_1, h_2, \dots, h_{k+1}\). By the induction hypothesis, we know that the \(k\) horses \(h_1, h_2, \dots, h_k\) are all the same color. Moreover, we know that the \(k\) horses labeled \(h_2, h_3, \dots, h_{k+1}\) are all the same color. Thus \(h_1\) is the same color as \(h_2\), which is the same color as \(h_{k+1}\), and so all \(k+1\) horses are the same color.

Thus since whenever \(k\) horses are the same color, we can conclude that \(k+1\) horses are the same color, it must be the case that all horses are the same color. \(\square\)
