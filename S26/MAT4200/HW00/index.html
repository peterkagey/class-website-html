In this class we will be writing lots of mathematics. As such, we will be learning about $\LaTeX$ (pronounced LAY-teck or LAH-teck) which is the most common way for professionals to typeset mathematics.1

We write our math in code, save them as .tex files, and compile them to get our final document (typically a PDF).

[2 pts] Exercise 3

(This is a problem that I give to my multivariable calculus students on their introductory assignment.)

In the next two questions we use vector calculus to calculate the bond angle of methane. We can determine the bond angle of methane ($\text{CH}_4$) by looking at the geometry of a regular tetrahedron, one of the five Platonic solids. (A regular tetrahedron can be constructed by gluing four equilateral triangles to each other by their edges.)

  1. Finding the center coordinate
    1. Verify that the coordinates $\mathbf{v}_1 = (0,0,0)$, $\mathbf{v}_2 = (2,0,0)$, and $\mathbf{v}_3 = (1, \sqrt{3}, 0)$, all in the $xy$-plane, give the vertices of an equilateral triangle by showing that the distance between any pair of points is equal to $2$.
    2. Find a point $\mathbf{v}_4 = (x,y,z)$ that is distance $2$ from each of the previous three points. (Hint: You can avoid square roots by finding a point a squared-distance $2^2 = 4$ from each point.)
    3. The center $\mathbf{c}$ of the regular tetrahedron can be found by taking the average of all of the vertices. Find the coordinates of $\mathbf{c}$ (simplify your values as much as possible).
  2. Calculating the bond angle
    1. A methane molecule can be modeled by centering the carbon atom at $\mathbf{c}$ and putting the four hydrogen atoms at $\mathbf{v}_1$, $\mathbf{v}_2$, $\mathbf{v}_3$, and $\mathbf{v}_4$. Use the coordinates you have found in the previous exercise to determine $\cos(\theta)$, the cosine of the bond angle of a methane molecule.
    2. Use a calculator to estimate the bond angle $\theta$ in degrees, to 3 decimal points. How does this compare to a Google search for "bond angle methane"?

[5 pts] Choice Exercise 5

(I'll try to include a programming problem as a choice exercise on each homework assignment. This is this week's.)

Consider the function $\mathbf{c}\colon [0,1] \to \mathbb{R}^2$ defined by \[\mathbf{c}(t) = \begin{pmatrix}\sin(\pi t)\\(t-1/2)^2\end{pmatrix}\]

  1. Write a computer program that implements this function, and evaluate it at $t=0$ and $t=1$. What is the distance between $\mathbf{c}(0)$ and $\mathbf{c}(1)$?
  2. Now plot the curve in $\mathbb{R}^2$. One way to do this is using WolframAlpha with the command parametric plot (sin(2pi t),(t-1/2)^2), t=0..1, but you can use any tool you want. Take a screenshot of the plot and include it in your submission with \includegraphics. (Ask in Discord, do a Google search, or ask a LLM if you're not sure how to include an image.)
  3. Let's numerically estimate the arc length with five points: write a program (and include it here with its output) to compute \[ |\mathbf{c}(1/4) - \mathbf{c}(0)| + |\mathbf{c}(1/2) - \mathbf{c}(1/4)| + |\mathbf{c}(3/4) - \mathbf{c}(1/2)| + |\mathbf{c}(1) - \mathbf{c}(3/4)|. \]
  4. Now write out the integral that computes arc length. Try to do this from first principles without looking up the arc length formula. Do you want to try to analytically evaluate this integral? (I don't.)
    If we numerically integrate using WolframAlpha we get: 
    4.06366877944870514169574927050545316148425707...
    Use a numerical integration tool of your choosing, and confirm that what you get agrees.
  5. Now write a program to estimate the arc length when sampling with $1000$ points, and include the program and output here. How many digits of agreement do you get with the WolframAlpha value? How many samples do you need to get one more digit of agreement?
