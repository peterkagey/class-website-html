Exercise 3 [7]

Given some curve in $\mathbb{R}^n$, we often want to parameterize it so that we move at unit speed. We can always do this in principle (unless the curve is pathological) but usually we can't describe it explicitly for two reasons: there are "usually" not explicit forms for indefinite integrals, and there are "usually" not explicit forms for inverse functions. This is a rare example where you can find the unit speed parameterization explicitly!

Let \(\mathbf{c}\colon[0,2\pi] \to \mathbb{R}^2\) be defined as \[ \mathbf{c}(t) = \begin{pmatrix} e^t\cos(t) \\ e^t\sin(t) \end{pmatrix} \] on the interval $t \in [0,2\pi]$.

  1. Compute (by hand) the derivative \(\mathbf{c}'(t)\), and use this to compute the arc length by hand.
  2. Let $s(t)$ be the function that describes the length of the curve traversed on the interval $[0,t]$. (In particular, $s(2\pi)$ should recover your result from above.) Explicitly describe $s(t)$.
  3. Let $t(s)$ be the inverse function of $s(t)$, which describes the time $t$ at which the particle has traveled a distance of $s$. Explicitly compute (by hand) $t(s)$, and describe the interval $I$ on which $t\colon I \to \mathbb{R}$ is defined.
  4. Use a computer to compute the derivative \[ \frac{d}{ds}\left(\mathbf{c}(t(s))\right), \] and further compute the magnitude of this vector, and confirm that it is constant. If you do this in Mathematica, I encourage you to wrap your expression in FullSimplify[..., Assumptions -> s > 0].

Choice Exercise 8 [10] (Programming)

Let's do a unit speed parameterization of the Bézier curve \[ \mathbf{c}(t) = (1-t)^2 \begin{bmatrix} 1 \\ 0 \end{bmatrix} + 2t(1-t) \begin{bmatrix} 1 \\ 1 \end{bmatrix} + t^2 \begin{bmatrix} 0\\ 1 \end{bmatrix} \] where \(t \in [0,1]\).   1. [2] First, use a computer to compute the arc length of this curve. Write down the integral, including the bounds and the integrand. (Show your work on how you determined the integrand, even if you used a computer.)
  2. [2] Let $s(t)$ be the distance you have traveled along the curve over the interval $[0,t]$. Convince yourself that there (probably) isn't a closed-form for $s(t)$, then determine the approximate value of $t$ such that $s(t) = 1$.
  3. [1] Now let $t(s)$ be the inverse function of $s(t)$. What interval is $s$ defined on? That is, what input values of $s$ make sense?

  5. [1] Numerically approximate the tangent vector of $\mathbf{c}(t(s))$ when $s = 1$, and numerically approximate the magnitude of the vector.
  6. [1] Write a function to numerically approximate the magnitude of the acceleration vector of $\mathbf{c}(t(s))$ at distance $s$. Call this function $\kappa(s)$. Estimate $\kappa(0.811)$.
