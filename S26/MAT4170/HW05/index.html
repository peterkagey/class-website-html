Choice Exercise 4 [≤9 + 5 + 5 + 6] (review/practice)
  1. [≤9] We define a normal subgroup of $(G,*)$ as one where $g * n * g^{-1} \in N$ for all $n \in N$ and $g \in G$. Do any subset of the following questions.
    1. Explain why the following are true for every subgroup $H \leq G$:
      1. For all $g \in G$ and $h \in H$, $g * h * g^{-1} \in G$
      2. For all $g \in H$ and $h \in H$, $g * h * g^{-1} \in H$
    2. Prove that if $G$ is abelian then all subgroups $H \leq G$ are normal.
    3. Prove that both the trivial subgroup $\{e\} \leq G$ and the group $G$ itself are normal subgroups of $G$.
    4. Let $G = D_6$, and recall that all elements of $D_n$ can be written of the form $r^k f^m$. Use the fact that $r^kf = fr^{-k}$ to show that $\langle r^2 \rangle$ is a normal subgroup of $D_6$.
    5. [2] Let $G = D_n$. Show that if $d$ is a divisor of $n$ then $\langle r^d \rangle$ is a normal subgroup of $D_n$. Hint: this is a generalization of the above part, so do that part first!
    6. Recall that $\{\pi \in S_4 \mid \pi(4) = 4\}$ is a subgroup of $S_4$. Prove that it is not a normal subgroup of $S_4$.
    7. [2] Prove that if $(G,*)$ is a group and $H$ is a subgroup. Prove that if ${[H : G] = 2}$ then $H$ is a normal subgroup of $G$.
  2. [4] On HW4, you played with the direct product of groups. \[ G_1 \times G_2 = \{(g_1, g_2) \mid g_1 \in G_1, g_2 \in G_2 \}, \] where the binary operation is applied componentwise.
    1. Prove that for every group $G$, $G \times \mathbf{1}$ is isomorphic to $G$ by constructing an explicit isomorphism, where $\mathbf{1}$ is the trivial group, which consists of only the identity.
    2. Prove that $G_1 \times G_2$ is isomorphic to $G_2 \times G_1$ by constructing an explicit isomorphism.
    3. Let $\mathbf{1} = \langle e \rangle \subseteq S_4$ be the trivial subgroup of $S_4$. Prove $D_5 \times \mathbf{1} \lhd D_5 \times S_4$.
    4. Prove that if $N_1 \lhd G_1$ and $N_2 \lhd G_2$, then $N_1 \times N_2 \lhd G_1 \times G_2$.
  3. Prove the following two statements about equivalence relations, which are relations that are reflexive, symmetric, and transitive.
    1. [3] Prove that group isomorphism is an equivalence relation.
    2. [2] Prove that an equivalence relation on a set partitions the set, which is to say that we can write a set as \[ S = S_1 \cup S_2 \cup \dots \cup S_n \] where each $S_i \subseteq S$, and $S_i \cap S_j = \emptyset$ if and only if $i \neq j$.
  4. [6] Let $G = \mathbb{Z} \times \mathbb{Z}$, which we will denote $G = \mathbb{Z}^2$. Consider the subgroup $H = \langle(1,2)\rangle$.
    1. Plot at least four elements of $H$ as points in the $xy$-plane, where $(1,2)$.
    2. Choose a coset $(x_1, y_1) + H$, and plot at least four elements of this coset.
    3. Choose another coset $(x_2, y_2) + H$, and plot at least four elements of this coset.
    4. Qualitatively describe the geometry of these cosets when plotted.
    5. Prove that $H \lhd G$.
    6. Prove that $G/H \cong \mathbb{Z}$ by describing an explicit isomorphism $\varphi$.
Choice Exercise 5 [≤12] (Number theory)

An earlier problem suggested that if $\gcd(k,n) = 1$ then $\langle r^k \rangle = C_n$. In this problem you will prove that this (and other group theory facts) follow from Bézout's identity, which states that if $k$ and $n$ are coprime (i.e. $\gcd(k,n)=1$), then there exist integers $a$ and $b$ such that $ak + bn = 1$. At the end, you will prove Bézout's identity.

  1. [1] Assuming Bézout's identity, explain how $r^{ak}=r \in C_n$ and thus $\langle r^k \rangle = C_n$.
  2. [2] Again assuming Bézout's identity, give an explicit element of ${\mathbb{Z}/k\mathbb{Z} \times \mathbb{Z}/n\mathbb{Z}}$ that has order $kn$, and use it to construct an isomorphism \[ \phi\colon \mathbb{Z}/k\mathbb{Z} \times \mathbb{Z}/n\mathbb{Z} \to \mathbb{Z}/kn\mathbb{Z}. \]
  3. [2] Show how the above isomorphism gives a (non-constructive) proof of the existence part of the Chinese Remainder Theorem which states that if $n$ and $k$ are coprime, and $\alpha$ and $\beta$ are any integers, then there exists an integer $m$ such that \[\begin{align*} m &\equiv \alpha \pmod n \\ m &\equiv \beta \pmod k. \end{align*}\]
  4. [4] Now we will prove Bézout's identity in the case that $\gcd(a,b) = 1$ where $a > b \ge 1$.
    1. We'll start by describing the basis of the Euclidean algorithm. Suppose without loss of generality that $a \geq b$ (since $\gcd(a,b) = \gcd(b,a)$), and show that $\gcd(a,b) = \gcd(a-b,b)$.
    2. Show that we can only continue this process for a finite number of steps before we reach $\gcd(1,1) = 1$.
    3. Now prove that we can inductively unwind the whole process in the following sense: if we can find integers $c_1$ and $c_2$ such that $c_1(a-b) + c_2(b) = 1$, then we can find integers $d_1$ and $d_2$ such that $d_1 a + d_2 b = 1$.
    4. Show that it works in practice by computing $\gcd(7,4) = 1$ using a series of moves like those described in part a, and unwind it to find integers $a$ and $b$ such that $7a + 4b = 1$. (An intermediate step might be that $4c_1 + 3c_2 = 1$ when $c_1 = -2$ and $c_2 = 3$.)
  5. [3] Write a program that takes in two positive integers $(k,n)$ and returns a pair $(a,b)$ such that $ak + bn = \gcd(k,n)$.
