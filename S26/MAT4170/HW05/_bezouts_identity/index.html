An earlier problem suggested that if $\gcd(k,n) = 1$ then $\langle r^k \rangle = C_n$. In this problem you will prove that this (and other group theory facts) follow from Bézout's identity, which states that if $k$ and $n$ are coprime (i.e. $\gcd(k,n)=1$), then there exist integers $a$ and $b$ such that $ak + bn = 1$. At the end, you will prove Bézout's identity.

  1. [1] Assuming Bézout's identity, explain how $r^{ak}=r \in C_n$ and thus $\langle r^k \rangle = C_n$.
  2. [2] Again assuming Bézout's identity, give an explicit element of ${\mathbb{Z}/k\mathbb{Z} \times \mathbb{Z}/n\mathbb{Z}}$ that has order $kn$, and use it to construct an isomorphism \[ \phi\colon \mathbb{Z}/k\mathbb{Z} \times \mathbb{Z}/n\mathbb{Z} \to \mathbb{Z}/kn\mathbb{Z}. \]
  3. [2] Show how the above isomorphism gives a (non-constructive) proof of the existence part of the Chinese Remainder Theorem which states that if $n$ and $k$ are coprime, and $\alpha$ and $\beta$ are any integers, then there exists an integer $m$ such that \[\begin{align*} m &\equiv \alpha \pmod n \\ m &\equiv \beta \pmod k. \end{align*}\]
  4. [4] Now we will prove Bézout's identity in the case that $\gcd(a,b) = 1$ where $a > b \ge 1$.
    1. We'll start by describing the basis of the Euclidean algorithm. Suppose without loss of generality that $a \geq b$ (since $\gcd(a,b) = \gcd(b,a)$), and show that $\gcd(a,b) = \gcd(a-b,b)$.
    2. Show that we can only continue this process for a finite number of steps before we reach $\gcd(1,1) = 1$.
    3. Now prove that we can inductively unwind the whole process in the following sense: if we can find integers $c_1$ and $c_2$ such that $c_1(a-b) + c_2(b) = 1$, then we can find integers $d_1$ and $d_2$ such that $d_1 a + d_2 b = 1$.
    4. Show that it works in practice by computing $\gcd(7,4) = 1$ using a series of moves like those described in part a, and unwind it to find integers $a$ and $b$ such that $7a + 4b = 1$. (An intermediate step might be that $4c_1 + 3c_2 = 1$ when $c_1 = -2$ and $c_2 = 3$.)
  5. [3] Write a program that takes in two positive integers $(k,n)$ and returns a pair $(a,b)$ such that $ak + bn = \gcd(k,n)$.
