[8 pts] Exercise 2

In this class we will be writing lots of mathematics. As such, we will be learning about $\LaTeX$ (pronounced LAY-teck or LAH-teck) which is the most common way for professionals to typeset mathematics.1

We write our math in code, save them as .tex files, and compile them to get our final document (typically a PDF).

  • Download this minimal $\LaTeX$ homework template.
  • Go to Overleaf,2 make an account if necessary, and start a new project.
  • Click on the "upload" button in the upper left corner of the screen, and upload the .tex file you downloaded. The upload button looks like this: .

If you haven't used $\LaTeX$ before, don't worry—we're going to develop that skill over the course of this term. Here are the first few steps for getting started.

  1. This first part of the problem will get you set up with the document that you will turn in.
    1. Edit the \title{...} of the document to "HW 0" and the \author{...} to your name. The document will load after you compile, which you can start by hitting the green "Compile" or "Recompile" button on the right pane. (I compile by using the shortcut ⌘+S on MacOS.)
    2. Under \maketitle, write \section*{Exercise 1} and briefly state what image you posted on Discord and how the process of finding the Gradescope assignment went.
    3. After Exercise 1, write \pagebreak (which starts a new page) and then on the next line, write \section*{Exercise 2} where you will put your responses to this question.
  2. You can do one of two things for this next part. If you're new to $\LaTeX$ you can just do the problems below. However, if you feel that you are already competent at writing $\LaTeX$ (i) say so here, (ii) skip the remaining sub-problems associated with this problem (iii) set a timer for 20 minutes, (iv) try to learn a new $\LaTeX$ skill (perhaps TikZ), and (v) post what you learned to the #hints-questions-tips Discord channel.
    1. Look at the $\LaTeX$ code below, and try to guess what it's going to look like when it compiles. (This is called display style math mode.) 
      The following integral converges: \[
  \int_{x=\varepsilon}^\infty \frac{1}{x^2} dx = \frac{1}{\varepsilon}.
\]
      Type (don't copy/paste) the above code in the section you just made. (You may need to scroll to the right to see the end of the code.)
      Did the result look exactly how you expected? Say how it was the same or different.
    2. Look at the following code and guess what you think it's going to do. Type (don't copy/paste) the following code in the section you just made. (This is called inline math mode.) 
      The following integral diverges: \(\int_1^\infty \frac{1}{x} dx.\)
      Describe how the output was similar to or different from what you expected.
    3. It's helpful to make lists for your homework, which you do using the enumerate environment like this: 
      \begin{enumerate}
  \item[2a.] (Move your response to Problem 2.2a here.)
  \item[2b.] (Move your response to Problem 2.2b here.)
  \item[2c.] Look! I made a list!
\end{enumerate}
    4. Think of a math symbol that you'd like to use, and go to Detexify, draw it, and include it here in math mode (e.g. \(...\)). Make sure to add it to your list you've already started \item[2d.].
    5. What's something else you'd like to know how to do? Tell us in the #hints-questions-tips Discord channel.
    6. After \end{enumerate}, add a \pagebreak to get ready for the next exercise.

[4 + 5 + 4 + 3] Choice Exercise 7

Do any subset of the following parts. Make sure to do all subparts within any attempted part.
  1. [4 pts] Prove or disprove the following facts about linear algebra:
    1. If $A \in \mathbb{R}^{2 \times 2}$ is a real-valued $2\times2$ matrix, and $I_2$ is the identity matrix, then there exists a matrix $B$ such that $AB = I_2$.
    2. For all matrices $A, B \in \mathbb{R}^{2 \times 2}$, $A + B = B + A$.
    3. For all pairs of matrices $A, B \in \mathbb{R}^{2 \times 2}$, $AB = BA$.
    4. If $A$ is an invertible matrix with inverse $A^{-1}$, then $A^{-1}$ is an invertible matrix with inverse $(A^{-1})^{-1} = A$. (Note that the $-1$ superscript is ``just notation,'' meaning we don't know a priori how multiplying matrices affects exponents, so it is not enough to simply show that $-1 \times -1 = 1$.)
  2. [5 pts]
    1. Suppose we have two $n \times n$ matrices $A$ and $B$ with entries $a_{ij}$ and $b_{ij}$ respectively. Let $C = AB$ be the result of matrix multiplication of $A$ and $B$. Describe $c_{ij}$ (the entry in the $i$th row and $j$th column of $C$) in terms of $a_{ij}$ and $b_{ij}$.
    2. Prove that matrix multiplication is associative. That is, for all $n \times n$ matrices $A$, $B$ and $M$, \[ (AB)M = A(BM). \]
  3. [4 pts] Give examples of the following:
    1. A $3 \times 3$ matrix $A$ where any two of the columns are linearly independent, but all three are not.
    2. A $2 \times 2$ matrix $B$ where $B \neq I_2$, but $B^3 = I_2$. (Here $I_2$ is the $2 \times 2$ identity matrix.)
    3. A $2 \times 2$ matrix $C$ where $C$ is not the zero matrix, but $C^2$ is the zero matrix.
    4. A $3 \times 3$ matrix $D$ such that matrix multiplication by $D$ sends
      1. the column vector $(1,0,0)^T$ to $(7,2,6)^T$,
      2. the column vector $(0,1,0)^T$ to $(4,1,7)^T$, and
      3. the column vector $(0,0,1)^T$ to $(9,1,9)^T$.
  4. [3 pts] Before starting this problem, find an online linear algebra textbook and grab a pen and paper (or a tablet).
    1. Start a timer for 30 minutes. While the timer is running, do any problems that look challenging and interesting.
    2. Take a photo or screenshot of your work and include it here using \includegraphics. (Feel free to ask for help on Discord or you can use any tools to get it working. Just make sure the image looks nice in your PDF.)
    3. Type a list of the ideas that you learned (or relearned).
